부분 PDF 이름 설정

기본 이름을 입력하면 모든 부분 PDF에 같은 규칙이 적용됩니다. 예를 들어 자바책을 입력하면 자바책_1~3.pdf, 자바책_4~8.pdf처럼 저장됩니다.

전체 다운로드를 누르면 입력된 모든 유효 구간을 모아 ZIP 파일 한 개로 다운로드합니다.

부분 PDF 설정

각 구간의 시작 페이지와 끝 페이지를 입력한 뒤 파일을 생성하세요. 필요한 만큼 구간을 추가할 수 있고, 전체 다운로드 버튼으로 ZIP 파일 하나를 만들 수 있습니다.

페이지 번호는 1부터 시작합니다. 예를 들어 10페이지 PDF에서 4와 6을 입력하면 4, 5, 6페이지가 포함된 새 PDF가 생성됩니다.
